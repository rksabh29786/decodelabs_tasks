# 🤖 Rule-Based AI Chatbot

A simple Rule-Based AI Chatbot developed in Python using **if-else decision-making** and a **continuous loop**. This project was created as **Project 1** for the **DecodeLabs Artificial Intelligence Internship (Batch 2026)**.

---

## 📌 Project Overview

This chatbot simulates a basic conversation by responding to predefined user inputs. Unlike machine learning chatbots, it follows explicit rules created using Python control flow statements.

The project demonstrates fundamental Artificial Intelligence concepts such as:

- Rule-based systems
- Decision making
- Control flow
- User interaction
- Continuous conversation loop

---

## 🚀 Features

- 👋 Responds to greetings (Hello, Hi, Hey)
- 😊 Answers "How are you?"
- 🤖 Introduces itself
- 🧠 Explains Artificial Intelligence
- 🐍 Provides information about Python
- 📅 Displays the current date
- 🕒 Displays the current time
- 😂 Tells a programming joke
- 💡 Gives motivational messages
- 🙏 Responds to thanks
- ❓ Help command to display available commands
- 🚪 Exit using **exit**, **bye**, or **quit**
- ⚠ Handles unknown questions gracefully
- ⌨ Handles empty user input
